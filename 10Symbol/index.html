es5六种数据类型：

  • undefined
  • null
  • Boolean
  • String
  • Number
  • Object
  • Symbol

第七种就是--Symbol

		let s = Symbol();
		typeof s;   //symbol

Symbol函数前面不能加new

		let s1 = Symbol('foo');
		let s2 = Symbol('bar');

		s1  //Symbol(foo)
		s2  //Symbol(bar)

		s1.toString()  //"Symbol(foo)"
		s2.toString()  //"Symbol(bar)"
