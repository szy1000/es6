es5六种数据类型：

  • undefined
  • null
  • Boolean
  • String
  • Number
  • Object
  • Symbol

第七种就是--Symbol

		let s = Symbol();
		typeof s;   //symbol

Symbol函数前面不能加new

		let s1 = Symbol('foo');
		let s2 = Symbol('bar');

		s1  //Symbol(foo)
		s2  //Symbol(bar)

		s1.toString()  //"Symbol(foo)"
		s2.toString()  //"Symbol(bar)"

Symbol的参数是对象的话，就会调用对象的toString()方法，生成一个值

		const obj = {
			toSting() {
				return 'abc'
			}
		}

		const sym = Symbol(obj)
		sym //Symbol(abc)

注意，Symbol函数的参数只是表示对当前 Symbol值的描述，因此相同参数的Symbol函数的返回值是不相等的。

		var sym1 = Symbol();
		var sym2 = Symbol();

		sym1 === sym2 //false

		var sym1_one = Symbol('baz');
		var sym2_two = Symbol('baz');
		
		s1 === s2 //false

Symbol 值不能与其他类型的值进行运算，会报错。但是，Symbol 值可以显式转为字符串。另外，Symbol 值也可以转为布尔值，但是不能转为数值。

		var sym_string = Symbol('My symbol');

		"your symbol is " + sym_string //TypeError
		'your symbol is $(sym_string)'

转化字符串

		var sym_to_string = Symbol('My symbol');
		String(sym_to_string);
		sym_to_string.toString();
